# Movie Review Platform

A simple web application where users can submit movie reviews and receive AI-based sentiment analysis.
This is a simple project where i am trying to integrate my ML models to a movie review website.
I am still working on this!!.

## Tech Stack

Frontend

- HTML
- CSS
- JavaScript

Backend

- FastAPI

Database

- PostgreSQL

Machine Learning

- Scikit-learn sentiment analysis model

## Features

- Submit movie reviews
- Star rating system
- Sentiment analysis (positive/negative)
- Store reviews in PostgreSQL
- Display reviews by movie
